# ✨ About The Project

**MathLab** is an interactive educational platform designed to make **Double Integration** intuitive and visual.

Instead of learning only through static formulas, this project allows users to:

* Visualize bounded regions dynamically
* Understand iterated integrals visually
* Explore inner & outer limits of integration
* Practice MCQs in an NTA-style environment
* Generate step-by-step mathematical solutions
* Learn multivariable calculus interactively
